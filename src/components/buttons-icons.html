Botón de ícono

Este tipo de botones proporciona una forma visualmente intuitiva de interactuar con una aplicación o un sitio web. El uso de iconos puede ayudar a ahorrar espacio en la interfaz de usuario y hacer que a los usuarios les resulte más fácil reconocer y comprender la acción asociada con el botón. Piense en el icono de la papelera para eliminar elementos, el icono de un lápiz para editar contenido o el icono de una lupa para la acción de búsqueda.

Botón icono simple
