Selects

Un componente de selección es un menú desplegable para mostrar opciones. Utilice el componente de opción cuando haya menos opciones en total (menos de 5). A continuación nuestro ejemplo de componente "select" que puede utilizar en su proyecto. El ejemplo viene en diferentes colores y estilos, por lo que puede adaptarlo fácilmente a sus necesidades.

Select simple

Utilice este versátil componente Select con 4 opciones disponibles y diseñado con Tailwind CSS para recopilar información del usuario. Este ejemplo se adapta perfectamente a los formularios para que los usuarios puedan elegir un dato.

                        
<div class="w-full max-w-sm min-w-[200px]">
    <div class="relative">
        <select
            class="w-full py-2 pl-3 pr-8 text-sm transition duration-300 bg-transparent border rounded shadow-sm appearance-none cursor-pointer placeholder:text-secondary-400 text-secondary-700 border-secondary-200 ease focus:outline-none focus:border-secondary-400 hover:border-secondary-400 focus:shadow-md">
            <option value="optionOne">Opción Uno</option>
            <option value="optionTwo">Opción dos</option>
            <option value="optionThree">Opción tres</option>
            <option value="optionFour">Opción cuatro</option>
        </select>
    </div>
</div>
