Tooltips

Un Tooltip es un pequeño elemento emergente que aparece cuando el usuario mueve el puntero del ratón sobre un elemento. Úselo cuando desee especificar información adicional sobre algo cuando el usuario mueve el puntero del ratón sobre un elemento.

Tooltip predeterminado

Puedes inicializar nuevo tooltip agregando el atributo de datos data-tooltip-target="{tooltipName}" al elemento desencadenador y al elemento que desea utilizar como información sobre herramientas. data-tooltip-target="{tooltipName}" data-tooltip="{tooltipName}"

Este es un Tooltip 
                            
<button data-ripple-light="true" data-tooltip-target="tooltip" class=" btn-primary btn-md">
    Mostrar Tooltip
</button>
<div data-tooltip="tooltip"
    class="absolute z-50 whitespace-normal break-words rounded-lg bg-black py-1.5 px-3 font-sans text-sm font-normal text-white focus:outline-none">
    Este es un Tooltip
</div>
