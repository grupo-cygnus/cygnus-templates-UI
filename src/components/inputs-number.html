Inputs

Los campos de entrada permiten a los usuarios introducir texto en una interfaz de usuario. Suelen aparecer en formularios y cuadros de diálogo. Los campos de entrada en móviles deben tener un tamaño de texto de al menos 16px para evitar el zoom automático en los navegadores móviles.

Input simple

Utilice este ejemplo de "input" para comenzar fácilmente. Es perfecto para formularios que requieren la entrada de texto, como el registro de usuarios, formularios de inicio de sesión o cualquier formulario en el que los usuarios necesiten ingresar un nombre de usuario.

                        
<input class="input" placeholder="Escribir aquí..." />

Input con label

Utilice este ejemplo de "input" con el texto de marcador de posición «Escribir aquí», guiando al usuario para que introduzca su nombre. La etiqueta se coloca encima del campo de entrada para obtener claridad sobre qué información se requiere.

                        
<label class="label-input">Label</label>
<input class="input" placeholder="Escribir aquí..." />

Input con label y texto de ayuda

Utilice este botón versátil que viene con fondo azul y texto blanco, esquinas redondeadas y un efecto de sombra al pasar el mouse, mostrando interactividad.

Este es un texto de ayuda para el usuario. 

                        
<label class="label-input">Label</label>
    <input class="input" placeholder="Escribir aquí..." />
<p class="hint-text">Este es un texto de ayuda para el usuario.</p>

Input con label interactivo

Utilice este botón versátil que viene con fondo azul y texto blanco, esquinas redondeadas y un efecto de sombra al pasar el mouse, mostrando interactividad.

                        
<div class="relative">
    <input class="input-interactive peer" placeholder="" />
    <label class="label-input">Escribir aquí...</label>
</div>
