Dropdown Menu

El menú desplegable muestra una lista de opciones en superficies temporales. Aparece cuando los usuarios interactúan con un botón, una acción u otro control. Cuando un usuario hace clic o pasa el cursor sobre el elemento principal, aparece el menú desplegable, que presenta una lista de acciones apiladas verticalmente.

Menú desplegable simple

Este ejemplo muestra un componente de menú desplegable simple y responsivo que incluye un botón para abrir el menú y una lista <ul> que representa el menú en sí.

Menú desplegable anidado

Utilice este ejemplo de componente si desea implementar una estructura de menú desplegable anidada. Puede hacerlo agregando un data-popover-target="nested-menu" atributo a uno de los elementos del menú, de modo que al seleccionar este elemento se abra otro menú adyacente al actual. Puede crear menús anidados o de varios niveles configurando el data-popover-nested="true" atributo de datos en el elemento activador del menú.
