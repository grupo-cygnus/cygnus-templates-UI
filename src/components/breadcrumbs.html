Breadcrumbs

Las migas de pan sirven como una ayuda visual eficaz, indicando la ubicación del usuario dentro de la jerarquía del sitio web, lo que las convierte en una gran fuente de información contextual para las páginas de destino. Además, las migas de pan facilitan la navegación a páginas de nivel superior.

Breadcrumb básico

Utilice este botón versátil que viene con fondo azul y texto blanco, esquinas redondeadas y un efecto de sombra al pasar el mouse, mostrando interactividad.

Breadcrumb a nivel de bloque

Breadcrumb puede ser un componente a nivel de bloque que obtiene todo el espacio disponible en una fila. Puede mostrar los breadcrumbs como un elemento a nivel de bloque utilizando la clase w-full.

  1. Inicio
  2. Centro de aplicaciones
  3. Aplicaciones 
                            
<nav aria-label="breadcrumb" class="w-full">
    <ol class="flex flex-wrap items-center w-full px-4 py-2 rounded-md bg-secondary-50">
        <li
            class="flex items-center text-sm transition-colors duration-300 cursor-pointer text-secondary-500 hover:text-secondary-800">
            <a href="#">Documentos</a>
            <span class="mx-2 pointer-events-none text-secondary-800">
                /
            </span>
        </li>
        <li
            class="flex items-center text-sm transition-colors duration-300 cursor-pointer text-secondary-500 hover:text-secondary-800">
            <a href="#">Componentes</a>
            <span class="mx-2 pointer-events-none text-secondary-800">
                /
            </span>
        </li>
        <li
            class="flex items-center text-sm transition-colors duration-300 cursor-pointer text-secondary-500 hover:text-secondary-800">
            <a href="#">Breadcrumbs</a>
        </li>
    </ol>
</nav>
