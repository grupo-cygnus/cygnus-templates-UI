Elementos Misceláneos

Esta sección reúne elementos visuales y funcionales que no encajan en las categorías principales del diseño. Aquí encontrarás loaders, avatares, íconos, imágenes y otros componentes diversos que complementan la experiencia de usuario. Estos elementos están diseñados para ser reutilizables y flexibles, adaptándose a diferentes contextos y necesidades en tu proyecto.

Cargando

Utilice indicadores dentro de los botones para indicar que una acción se está procesando o llevando a cabo en ese momento. También se puede cambiar el texto del elemento indicador y utilizar el texto del botón según sea necesario..

                         
                            
<!-- Solo icono -->
<button type="button" class="px-0 btn-primary btn-lg">
    <span class="spin-loading">
        <span class="sr-only">Cargando...</span>
    </span>
</button>

<!-- Icono dentro de botón -->
<button type="button" class="px-0 btn-primary btn-lg">
    <span
        class="spin-loading"
        role="status" aria-label="loading"></span>
    Cargando
</button>

Avatar con iniciales

Esto se puede utilizar en ausencia de una foto de perfil. Se pueden utilizar las iniciales del usuario. Por ejemplo, para Juan Pérez puede mostrar JP.

    <span class="font-medium text-secondary-600">JP</span>
</div>
