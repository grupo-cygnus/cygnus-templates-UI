Alertas

Una alerta muestra un mensaje breve e importante que atrae la atención del usuario sin interrumpir su tarea. Utilice este elemento cuando necesite mostrar mensajes muy importantes a los usuarios o cuando necesite un contenedor estático persistente que pueda cerrarse mediante acciones del usuario.

Alerta simple

Este ejemplo de alerta con texto blanco y fondo gris oscuro crea una relación de alto contraste que mejora la legibilidad y atrae la atención.

Una simple alerta rellena para mostrar el mensaje información.

    Una simple alerta rellena para mostrar el mensaje información.

        class="flex items-center justify-center transition-all w-8 h-8 rounded-md text-primary-700 hover:bg-white/10 active:bg-white/10 absolute top-1.5 right-1.5"

            </path>

    </button>
</div>
