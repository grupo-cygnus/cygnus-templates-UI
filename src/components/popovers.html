Popovers

Popopovers sirven para mostrar información adicional, contexto o controles relacionados con un elemento o una acción. Los elementos emergentes se pueden activar mediante diversas interacciones del usuario, como hacer clic en un botón, pasar el cursor sobre un enlace de texto o centrarse en un campo de entrada. Los elementos emergentes son útiles para crear una interfaz de usuario más limpia y ordenada, ya que ocultan información secundaria u opciones hasta que se las necesita. Suelen utilizarse para información sobre herramientas, menús desplegables, menús de opciones, mensajes de validación de formularios y más.

    Mostrar Popover
</button>
<div data-popover="popover"

</div>
