Colapsable predeterminado

Usa el componente colapsable CSS Tailwind para permitir al usuario mostrar y ocultar secciones de contenido relacionado en una página. Hay muchas formas de utilizar este componente, como mostrar una lista de preguntas frecuentes, mostrar varios menús y submenús, mostrar las ubicaciones de una empresa en particular, etc.

Utilice este colapsable hecho en Tailwind. Puedes utilizarlo para acordeón, elementos plegables y mucho más. 

                        
<div class="grid min-h-[140px] w-full place-items-center overflow-x-scroll rounded-lg p-6 lg:overflow-visible">

    <button data-collapse-target="collapse" class="btn-primary" type="button">
        Abrir Colapsable
    </button>
    <div data-collapse="collapse"
        class="block w-full h-0 overflow-hidden transition-all duration-300 ease-in-out basis-full">
        <div class="relative flex flex-col w-8/12 mx-auto mt-4 bg-white border rounded-lg shadow-sm border-slate-200">
            <div class="p-4">
                <p class="font-light text-slate-600">
                    Utilice este colapsable hecho en Tailwind. Puedes utilizarlo para acordeón,
                    elementos plegables y mucho más.
                </p>
            </div>
        </div>
    </div>
</div>
