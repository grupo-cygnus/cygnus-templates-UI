Uso de SVGs

Los SVGs son elementos gráficos escalables que se pueden personalizar fácilmente para adaptarse a diferentes necesidades. En este proyecto, se siguen estas prácticas recomendadas para garantizar consistencia y personalización:

Prácticas Recomendadas

  • Aplicar las clases w-5 y h-5 para definir el tamaño de los íconos.
  • Utilizar el atributo stroke="currentColor" para heredar el color del texto padre.
  • Evitar incluir los atributos width y height directamente en los SVGs.
  • Importar los SVGs como componentes o directamente en el HTML según la necesidad.

Ejemplo de Uso

A continuación, se muestra un ejemplo de cómo usar un SVG correctamente:

                            
<svg class="w-5 h-5" xmlns="http://www.w3.org/2000/svg" fill="none" viewBox="0 0 24 24" stroke="currentColor">
<path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 4v16m8-8H4" />
</svg>

Este ejemplo crea un SVG con un tamaño de 20px (definido por w-5 y h-5) y hereda el color del contenedor padre.

Ejemplo con Cambios de Tamaño

Puedes ajustar el tamaño del SVG usando clases utilitarias de TailwindCSS:

w-4 h-4

w-6 h-6

w-8 h-8

Notas Adicionales

Para un mejor rendimiento, se pueden optimizar los SVGs antes de incluirlos en el proyecto usando herramientas como SVGO.
