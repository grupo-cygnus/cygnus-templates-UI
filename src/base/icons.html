Colores UI

Nuestro sistema de diseño aprovecha un útil conjunto de estilos de color como punto de partida perfecto para cualquier marca o proyecto. En lo que respecta al color, el contraste es fundamental para garantizar la legibilidad del texto.

Opciones para el Uso de Iconos SVG

Existen tres formas principales de integrar iconos SVG en nuestros proyectos. Aquí te mostramos cómo funciona cada una y sus ventajas/desventajas.

1. SVG Inline

Usamos el SVG directamente en el HTML, permitiendo un control total sobre su color y tamaño a través de clases Tailwind. 

                                        <div class="text-blue-500">
                                            <svg class="w-6 h-6" fill="currentColor" viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg">
                                                <path d="..."></path>
                                            </svg>
                                        </div>

2. SVG como Imagen (`<img>`)

Los SVG se integran como imágenes externas referenciadas, útil para casos donde no necesitamos cambiar colores o estilos dinámicos. 

                                        <img src="assets/icons/check.svg" alt="Check Icon" class="w-6 h-6">

3. SVG como Background en CSS

Usamos el SVG como una propiedad `background-image` en CSS. Ideal para estilos decorativos o iconos que no cambian dinámicamente. 

                                        .icon {
                                            width: 1.5rem;
                                            height: 1.5rem;
                                            background-image: url('assets/icons/check.svg');
                                            background-size: contain;
                                            background-repeat: no-repeat;
                                        }
